Mission 2

Contexte : Pour cette mission qui avait pour nécéssité de travailer en équipe de deux, nous avaons choisis de travailler en équipe avec mon camarade Maxime. Ce projet est toujours réalisé dans le cadre de l'application GSB qui est un laboratoire pharmaceutique fictif, généralement utilisée comme exemple dans des cas d'étude ou des simulations en gestion d'entreprise pour les développeurs.

Attentes : La mission consiste à participer à l'évolution d'un site Web qui exploite les données de l'organisation. L'objectif étant de crée une session gestionnaire qui pourra permettre de se connecter avec un interface spécial gestionnaire. Cette interface apportera la possibilité d'afficher la liste des visiteurs, d'en ajouter, d'en supprimer ainsi que d'apporter la possibilité de modifier les informations des visiteurs directement dans la sesion gestionnaire. La possibilité de pouvoir télécharger la liste des visiteurs sous format pdf est aussi demandé. Vous trouverez si dessous la répartition de taches entre Maxime et moi, tout comme les capture de nos résultats.
